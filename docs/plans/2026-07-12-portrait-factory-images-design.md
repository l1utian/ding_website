# 竖版工厂图片展示设计

## 问题

工厂画廊使用固定横向画布和 `object-fit: cover`。第 3、4 张图片是竖版，当前只显示图片中间区域，货架上下内容被大量裁切。

## 方案

- 为共享工厂图片增加明确的 `fit` 元数据，横图使用 `cover`，两张竖图使用 `contain`。
- 横图保持现有铺满和缓慢放大效果。
- 竖图在中央完整显示并关闭放大效果；底层使用同一图片的弱化背景填充横向空间，前景原图保持清晰。
- 竖图缩略图使用 `contain`，让用户能看出完整构图。
- 画廊尺寸保持固定，切换横竖图时页面不跳动。

## 验证

- 数据测试确认两站共用相同图片元数据，且只有第 3、4 张使用 `contain`。
- 运行 React 检查、完整测试、类型检查和两个站点构建。
- 在桌面和手机尺寸下检查第 3、4 张截图，确认原图完整、清晰且控件不重叠。
